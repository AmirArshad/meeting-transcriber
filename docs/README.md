# Meeting Transcriber Documentation

This directory contains all documentation for the Meeting Transcriber project.

## Directory Structure

### `/design/`
Implementation plans and design documents for features and fixes:
- [AUDIO_DEVICE_COMPATIBILITY_PLAN.md](design/AUDIO_DEVICE_COMPATIBILITY_PLAN.md) - Complete plan for Bluetooth/multi-device audio support
- [BLUETOOTH_FIX_SUMMARY.md](design/BLUETOOTH_FIX_SUMMARY.md) - Summary of Bluetooth headset audio fix
- [FIRST_TIME_UX_IMPROVEMENTS.md](design/FIRST_TIME_UX_IMPROVEMENTS.md) - First-time user experience enhancements
- [INSTALLER_PERFORMANCE_IMPROVEMENTS.md](design/INSTALLER_PERFORMANCE_IMPROVEMENTS.md) - Installer optimization plans

### `/features/`
User-facing feature documentation

### `/development/`
Developer guides and development documentation

### `/internal/`
Internal documentation and notes

### Root-level docs
- [MEETING_TRANSCRIPTION.md](MEETING_TRANSCRIPTION.md) - Meeting transcription guide
- [ROADMAP.md](ROADMAP.md) - Project roadmap
- [TRANSCRIPTION_GUIDE.md](TRANSCRIPTION_GUIDE.md) - Transcription usage guide

## For Developers

When creating new documentation:
- **Design documents** (implementation plans, architecture) → `/design/`
- **Feature documentation** (user guides, how-tos) → `/features/`
- **Development guides** (setup, contributing, APIs) → `/development/`
- **Internal notes** (meeting notes, decisions) → `/internal/`
