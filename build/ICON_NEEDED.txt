# Icon File Needed

This directory needs an `icon.ico` file for the installer.

## Current Status
Using placeholder icon (GitHub Actions workflow creates one automatically)

## To Add Your Own Icon

1. Create or obtain a 256x256 PNG icon of a microphone
2. Convert to ICO format using:
   - Online: https://convertio.co/png-ico/
   - Windows: Use IcoFX or similar tool
   - PowerShell: See build-release.yml workflow for conversion script

3. Save as `build/icon.ico`
4. Commit to git: `git add -f build/icon.ico`

## Recommended Design
- **Theme:** Microphone or waveform
- **Size:** 256x256 (multi-resolution ICO)
- **Style:** Modern, clean, professional
- **Colors:** Blues/teals to match UI theme
